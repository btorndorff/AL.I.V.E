# AL.I.V.E Frontend
This is the frontend of our alive webapp that uses react.js. 
## Getting Started
1. Clone the repo into your local machine:
```
git clone https://github.com/btorndorff/alive-frontend.git
```
2. install the dependencies:
```
npm install
```
3. Start the webapp:
```
npm start
```
